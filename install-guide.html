⭐ 강력 추천

OpenAI API 추천

GPT 모델 기반 — 자연스러운 한국어 카피, 정확한 문장 구조, 더 풍부한 결과물

📝 발급 절차
  1. 1 platform.openai.com 접속 후 회원가입 (구글 계정 가능)
  2. 2 전화번호 인증 진행
  3. 3 Billing 메뉴에서 결제수단 등록 후 최소 5달러 충전 (약 7,400원, 환율 따라 변동) — Pay-As-You-Go 방식
  4. 4 API Keys 페이지 접속
  5. 5 "Create new secret key" 버튼 클릭
  6. 6 키 이름 입력 후 생성 → 키 즉시 복사 후 안전한 곳에 보관 (창 닫으면 다시 못 봄!)
🔗 OpenAI API 키 발급 페이지로 이동 →

INSTALLATION

설치 방법

아래 단계를 순서대로 따라하시면 5분 안에 설치 완료됩니다.

1

설치 파일 다운로드

GitHub 릴리즈 페이지에서 ShortsRemixer.Setup.3.0.1.exe 파일을 클릭해 다운로드합니다.
용량은 약 73.5MB입니다.

2

다운로드 완료 확인

브라우저 우측 상단에서 "ShortsRemixer.Setup.3.0.1.exe · 완료" 알림을 확인하고 파일을 클릭해 실행합니다.

3

Windows 보호 경고에서 "추가 정보" 클릭

실행 시 "Windows의 PC 보호" 경고가 뜹니다. 이는 정식 서명되지 않은 앱에 뜨는 일반적인 메시지입니다.
좌측 하단의 "추가 정보" 텍스트를 클릭해주세요.

💡 안전합니다! Windows는 알려지지 않은 신규 앱에 자동으로 이 경고를 표시합니다.
4

"실행" 버튼 클릭

앱 정보가 표시되면 좌측의 "실행" 버튼을 클릭해 설치를 진행합니다.

5

설치 옵션 선택

"이 컴퓨터를 사용하는 모든 사람(모든 사용자)"을 선택하고 "다음"을 클릭합니다.

6

설치 위치 선택

기본 설치 폴더 C:\Program Files\ShortsRemixer를 그대로 두고 "설치" 버튼을 클릭합니다.
필요 디스크 공간은 약 256MB입니다.

7

바탕화면 아이콘 더블클릭으로 실행

설치가 완료되면 바탕화면에 "ShortsRem..." 아이콘이 생성됩니다.
더블클릭해 앱을 실행해주세요.

8

회원가입 → 자동 로그인

앱이 실행되면 회원가입 화면이 나타납니다. 다음 정보를 입력해주세요:

  • 초대 코드 — 이메일로 받은 초대 코드를 입력하세요.
    체험판/월정액/평생 코드를 입력해주세요.
  • 아이디 — 3~30자
  • 이메일 — 본인 이메일
  • 비밀번호 — 6자 이상

"가입하기" 버튼을 누르면 자동으로 로그인되어 바로 사용 가능합니다!
